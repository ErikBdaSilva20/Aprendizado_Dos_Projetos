# 📁 Aprendizado de Projetos (Projects Learning)

Bem-vindo à seção de estudos baseada em projetos práticos! Aqui, cada pasta representa um projeto real ou boilerplate onde aplico a metodologia **LearnAgent** para documentar o aprendizado em contexto.

Os projetos adicionados aqui são aqueles em que tenho dificuldades em entender ou aplicar os conceitos, ou aqueles que quero aprofundar meu conhecimento.

## 📌 Objetivo

Diferente de estudos isolados de sintaxe, esta pasta foca em como as tecnologias se conectam na construção de aplicações complexas e escaláveis.

## 🛠️ O que aprendemos aqui?

Nos estudos baseados em projetos, focamos em:

- **Integração**: Como o Frontend fala com o Backend.
- **Arquitetura**: Como organizar pastas (DDD, Clean Architecture).
- **Automações de IA**: Fluxos n8n, APIs de mensageria, Docker e infra.
- **Fluxos Reais**: Autenticação, Pagamentos, Dashboards.
- **Resolução de Problemas**: Debugging e decisões técnicas.
